# Podrabotai 📱

Android приложение для поиска вакансий и работников.

## 🚀 Возможности

- Авторизация и регистрация  
- Просмотр вакансий  
- Публикация и редактирование объявлений  
- Фильтрация объявлений  
- Определение города по GPS  
- Чаты между пользователями  
- Профиль пользователя  
- Избранное  
- История просмотра  

## 🛠 Технологии

- Kotlin  
- Jetpack Compose  
- Clean Architecture  
- Hilt  
- Retrofit  
- Coroutines  
- Navigation Compose
